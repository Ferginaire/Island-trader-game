
To import the source code into a new Eclipse project,
	1. open Eclipse
	2. Create a new project by going file > new > Java project. 
	3. Name it and click finish. Don't create the module.
	4. Right click the source folder and choose import > general > File system
	5. Navigate to the src file, click select folder, make sure all of the class files are ticked and click finish
	6. Run the BasicUI class because that's where main is

To take the JAR file and run it
	1. Ensure Java 14 is installed
	2. Open a terminal window and cd to the folder that contains the JAR file
	3. type in and run "java -jar Fgr28_Obn11_IslandTrader.jar"
